# PowerShell

## Forensics
```
Find-Module -Name *Forensic*
Install-Module -Name PowerForensicsv2
Import-Module PowerForensicsv2
Get-Command -Module PowerForensicsv2
Get-ForensicNetworkList
Get-ForensicFileRecord -path c:\windows\system32\cmd.exe
Get-ForensicFileSlack -path c:\windows\system32\cmd.exe
```
